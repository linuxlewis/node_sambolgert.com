My work

This site

This site was my first foray into Node.js. In addition to Node, I used the Express web framework to power the site and designed it on top of Twitter's Bootstrap. The inspiration for my design came from sites made with about.me. Its hosted behind an Nginx proxy on Webfaction. The source code is available on Github

Se.cret.ly

Secretly is a web application that creates one time short links for links, images, and text. The idea came to me when I was chatting with a friend and didn't have a secure way to send them an image. Email wasn't an option, and other sites like imgur were too public. I designed and developed the entire application. It is powered by Ruby on Rails and Amazon Web Services and hosted in the cloud on Heroku.

Outloudio

Outloudio is a web application that creates a personal news audio-cast from articles found around the web. Outloudio was the product of the weekend long competition: Chicago Startup Weekend. I grouped up with other entreupeneurs to create a working product and business plan by the end of the weekend. I was the primary developer and coded and designed the entire backend system with Python and Django. Outloudio won "Best Technical Demo". It is hosted on my Webfaction server with Apache and mod_wsgi.

My Blog

In a true open source style my blog is based of a fork of the Ruby on Rails blog engine Enki. The design is based off my previous wordpress blog. You can see my fork of the project on Github. My blog is hosted seperately from this site in the cloud on the Heroku application platform.

UWSP Sustainable Commute Project

This website was made for the Sustainable Commute Project at my alma mater UW-Stevens Point. I was the architect for this application and designed it based on the specifications given to me by the primary researcher. I was responsible for recruiting additional members of the development team and outsourcing tedious work.The website explains the goals of the project and allows community members to take a "Commuting Survey". The interactive survey utilizes Google Maps to get the users route to and from UW-Stevens Point. The data gathered from this application will be used to redesign city road ways and reduce the community's impact on the environment. I coded the site with the Web2py web framework and its hosted on Google App Engine.

Out in SP

Out in SP is an Android Application that allows users to see a daily list of the bar specials of bars in Stevens Point, WI. Users can connect with each other over Facebook based on the bar they are currently at. I developed and designed the entire application for the 1st annual Noel Group Mobile Application Contest. My application took 1st place. It is currently available in the Android Market. The backend is powered by MySQL and PHP5 and hosted on my Webfaction server.
